## 🚀 Về mình

Mình là sinh viên IT yêu thích xây dựng các sản phẩm thực tế, đặc biệt là các hệ thống web backend, API, ứng dụng realtime và các dự án có áp dụng AI.

Hiện tại mình đang tập trung vào:

- Xây dựng REST API với NestJS, Prisma, PostgreSQL
- Phát triển frontend với React, TypeScript, TailwindCSS
- Làm việc với Git, GitHub, Docker, Railway, Vercel
- Nghiên cứu AI cho xử lý âm thanh, ghi biên bản cuộc họp và STT
- Xây dựng các dự án cá nhân và đồ án học tập có tính ứng dụng thực tế

## 📌 Dự án nổi bật

### 🔹 DoveRx — Medical Social Network & Forum

Nền tảng mạng xã hội & diễn đàn y tế kết nối người dùng quan tâm sức khỏe với bác sĩ/sinh viên y khoa, hỗ trợ hỏi đáp, chia sẻ kiến thức, tư vấn online và quản lý hồ sơ y tế cá nhân.

- Repo: https://github.com/vietTNT/DoveRx-frontend
- Demo: https://doverx.vercel.app

**Tech stack:** React.js, JavaScript, CSS, HTML (Frontend)

Chức năng chính:

- Q&A, chia sẻ kiến thức và tương tác cộng đồng
- Real-time Chat & Notifications (WebSocket)
- Auth & Security: JWT/OAuth2, refresh-token flow (Axios interceptors)
- Phân quyền User/Doctor (RBAC) và luồng đăng ký Doctor

---

### 🔹 Job Platform - IT Recruitment System

Hệ thống tìm kiếm và tuyển dụng việc làm IT, hỗ trợ người tìm việc, nhà tuyển dụng và quản trị viên.

**Tech stack:** NestJS, Prisma, PostgreSQL, React, TypeScript, TailwindCSS

Chức năng chính:

- Đăng nhập / đăng ký người dùng
- Quản lý hồ sơ ứng viên
- Quản lý công ty và nhà tuyển dụng
- Đăng tin tuyển dụng
- Ứng tuyển công việc
- Bookmark công việc
- Quản trị người dùng, công ty, ngành nghề

---

### 🔹 ViN Code Editor

Ứng dụng lập trình cộng tác realtime, kết hợp ý tưởng giữa VS Code và Google Docs.

**Tech stack:** NestJS, React, TypeScript, WebSocket, PostgreSQL, Electron

Chức năng chính:

- Quản lý workspace và project
- Chỉnh sửa code realtime
- Đồng bộ thay đổi giữa nhiều lập trình viên
- Quản lý lịch sử chỉnh sửa
- Tích hợp terminal
- Hướng đến desktop app bằng Electron

---

### 🔹 AI Meeting Minutes

Hệ thống chuyển âm thanh cuộc họp thành văn bản, tách người nói, dịch và tóm tắt nội dung.

**Tech stack:** Python, FastAPI, Faster-Whisper, Pyannote, Ollama, Qwen

Chức năng chính:

- Upload file ghi âm
- Chuyển giọng nói thành văn bản
- Tách người nói
- Dịch nội dung
- Tóm tắt cuộc họp bằng AI
- Trích xuất công việc cần làm
